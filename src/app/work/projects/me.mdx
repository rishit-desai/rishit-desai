## Overview

This project focused on creating a dashboard to manage and govern minor enhancements. The dashboard provides reporting capabilities and leverages machine learning to provide smart insights into the enhancement process.

## Key Features

-   **Enhancement Tracking**:  Tracks minor enhancements from inception to deployment.
-   **Reporting and Analytics**:  Provides detailed reports and analytics on enhancement progress and status.
-   **Machine Learning Insights**:  Uses machine learning to identify trends, predict outcomes, and provide recommendations.
-   **Governance Tools**:  Supports governance processes and ensures compliance.

## Technologies Used

-   Python
-   QuickBase
-   GitHub Actions
-   Machine Learning

## Challenges and Learnings

Integrating machine learning models to provide accurate and actionable insights was a significant challenge. I gained valuable experience in applying machine learning to improve business processes.

## Outcome

The dashboard has improved the management and governance of minor enhancements, leading to better decision-making and increased efficiency.
