## Overview

This project developed a system to analyze the sentiment of tweets in real-time. It utilizes the Word2Vec embedding model and a custom deep learning classifier to categorize tweets as positive, negative, or neutral.

## Key Features

-   **Real-time Tweet Analysis**:  Classifies the sentiment of tweets as they are posted.
-   **Sentiment Classification**:  Categorizes tweets into positive, negative, or neutral sentiment.
-   **Word Embedding**:  Uses the Word2Vec model to represent words as vectors.

## Technologies Used

-   Python
-   TensorFlow
-   Flask
-   React.js

## Challenges and Learnings

Building an accurate and efficient sentiment analysis model that can handle the nuances of language in tweets was a complex task. I gained experience in natural language processing and building real-time applications.

## Outcome

The project provides a tool for analyzing public opinion on Twitter, which can be valuable for various applications such as market research or social media monitoring.
