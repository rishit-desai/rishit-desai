## Overview

This personal project aimed to develop a machine translation tool capable of translating between several European languages and English.

## Key Features

-   **Multi-language Translation**:  Translates phrases and sentences from French, Spanish, Portuguese, German, and Italian to English.
-   **User-friendly Interface**:  Provides a simple and intuitive interface for users to input and translate text.

## Technologies Used

-   Python
-   TensorFlow
-   Deep Learning
-   Natural Language Processing
-   Streamlit

## Challenges and Learnings

Training accurate and efficient machine translation models was a complex task. I learned a great deal about NLP techniques and model optimization.

## Outcome

The project successfully demonstrates the application of deep learning for machine translation, providing a functional tool for language translation.
