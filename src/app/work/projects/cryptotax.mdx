## Overview

This project involved developing a website for CryptoTax, a company specializing in cryptocurrency tax management. The website includes an admin page to manage its content dynamically.

## Key Features

-   **Website Development**:  Created a website to provide information and services related to cryptocurrency taxes.
-   **Admin Page**:  Developed an admin interface for performing CRUD (Create, Read, Update, Delete) operations on the website's dynamic components.

## Technologies Used

-   React.js
-   Next.js
-   Firebase
-   AWS Lambda

## Challenges and Learnings

Building a website with dynamic content and a secure admin interface was a key challenge. I gained experience in developing modern web applications with React.js and Next.js, and using Firebase and AWS Lambda for backend functionalities.

## Outcome

The website provides CryptoTax with an online presence to reach clients and manage their services effectively.
